# Dashboard PyME - Analítica Ligera

Un proyecto modular en React + Material UI + Recharts que muestra métricas clave para pequeñas y medianas empresas (PyMEs). 
Este dashboard es parte de un portafolio orientado a Solution Architecture y DevOps, con enfoque en claridad, modularidad y narrativa técnica.
